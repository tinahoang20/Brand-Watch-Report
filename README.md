# Brand Watch Report - Ogilvy Entrance Test

## Introduction

This repository contains the project submission for the Ogilvy Entrance Test focusing on conducting a Brand Watch report for three brands in the F&B industry: Coke, Pepsi, and Fanta. The goal of this project is to analyze their social media presence, content strategies, and performance metrics over the timeframe from January to March 2025.

## Assignment Details

### Objective

To conduct a comprehensive social media audit on the following brands:

- **Coca-Cola**
- **Pepsi**
- **Fanta**

### Research Scope

**Data Collection:**

- Gather data from social media channels (Facebook, Instagram, Twitter).
- Collect metrics such as page followers, number of interactions (likes, comments, shares), and engagement rates.

**Content Analysis:**

- Categorize content angles, pillars, or messages used by each brand (e.g., advertising, promotional campaigns, user-generated content).
- Analyze content performance based on engagement metrics, reach, and impressions.

**Strengths and Weaknesses:**

- Provide an evaluation of the strengths and weaknesses of each brand's social media presence based on the data collected.

**Key Learnings:**

- Summarize observations and derive actionable insights for the client to enhance their own content strategies.

## Data Collection

### Data Sources

1.  **Google Trends**:
    *   Used to compare the search interest and trend analysis for each brand over time.
    *   ![Google Trends](google_trends_screenshot.png)
2.  **Social Searcher**:
    *   Used to gather mentions and conversations around each brand on social media platforms.
3.  **Manual Collection**:
    *   Directly collecting data from the official social media pages of each brand.

### Metrics Collected

| Metric                     | Description                                                                       |
| -------------------------- | --------------------------------------------------------------------------------- |
| Followers                  | Number of followers on each social media platform.                                |
| Engagement Rate            | Total interactions (likes, comments, shares) divided by the number of followers.  |
| Post Frequency             | Number of posts per week/month.                                                  |
| Content Type               | Types of content posted (e.g., images, videos, links).                              |
| Sentiment                  | Overall sentiment (positive, negative, neutral) of mentions and comments.          |

## Strengths and Weaknesses

### Coca-Cola

*   **Strengths**:
    *   Strong brand recognition and heritage.
    *   Effective use of seasonal campaigns.
*   **Weaknesses**:
    *   Lower engagement rate compared to competitors.
    *   Less interactive content.

### Pepsi

*   **Strengths**:
    *   High engagement rate with video content.
    *   Strong association with entertainment and sponsorships.
*   **Weaknesses**:
    *   Less consistent brand messaging.
    *   Lower follower count compared to Coca-Cola.

### Fanta

*   **Strengths**:
    *   Highest engagement rate with interactive content.
    *   Effective youth engagement strategies.
*   **Weaknesses**:
    *   Lower brand recognition compared to Coca-Cola and Pepsi.
    *   Limited content diversity.

## Key Learnings

### Actionable Insights

1.  **Enhance Interactive Content**:
    *   All brands should increase the use of interactive content to boost engagement rates.
2.  **Diversify Content**:
    *   Coca-Cola should diversify its content to include more engaging and interactive posts.
3.  **Consistent Messaging**:
    *   Pepsi should focus on creating more consistent brand messaging.

### Strategic Recommendations

1.  **Leverage User-Generated Content**:
    *   Encourage users to create and share content featuring the brands to increase engagement and authenticity.
2.  **Run Targeted Campaigns**:
    *   Design campaigns that target specific demographics and interests to improve relevance and engagement.
3.  **Monitor Social Sentiment**:
    *   Continuously monitor social media sentiment to identify issues and opportunities in real-time.
